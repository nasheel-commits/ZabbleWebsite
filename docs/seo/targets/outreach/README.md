# Outreach Kit — Zabble link-building & digital PR (owned by S04)

Everything needed to run authority-building outreach the moment the owner asks
(B5/B6 list at the end). Grounded in the competitor link-asset analysis in
`../../audits/04-offpage-local.md` §5 and the SA market evidence in
`../../_evidence/04/`. Zabble is a **new domain** — this is a slow, quality-first
programme, not a link blitz.

## What's in here

| File | Purpose |
|------|---------|
| `prospect-list.md` | Prioritised targets by opportunity type, with angle + suggested anchor. |
| `angles.md` | Pitch angles tied to specific Zabble systems/case stories. |
| `email-templates.md` | Ready-to-send templates per angle (placeholders only). |
| `tracker.csv` | The live outreach tracker — one row per prospect, status pipeline. |
| `anchor-guidance.md` | Anchor-text distribution rules (penalty-safe for a young domain). |
| `velocity-plan.md` | Month-by-month link-velocity plan + monthly monitoring once B5 lands. |

## How to run it (operating loop)

1. **Foundation first.** Don't pitch editorial links until the citation/entity base
   exists (GBP + LinkedIn + Crunchbase + Clutch). See `../local-kit/citation-checklist.csv`.
2. **Lead with the asset, not the ask.** Every pitch references a real, useful thing —
   most often the `/insights/south-african-operations-software-landscape` reference,
   or a specific client outcome. Outreach without a reason to exist gets ignored.
3. **One prospect, one angle, one anchor.** Pick the angle from `angles.md` that fits
   the prospect; pick the anchor per `anchor-guidance.md`. Log it in `tracker.csv`
   *before* you send.
4. **Personalise the first two lines.** Reference something the recipient actually
   wrote/published. Templates are scaffolding, not spray.
5. **Respect velocity.** `velocity-plan.md` caps how many editorial links to chase per
   month so the profile grows believably.
6. **Never buy links, never incentivise reviews.** One earned editorial link beats ten
   paid directory drops, and beats a Google penalty by infinity.

## Status / owner asks (consolidated)

This kit is **execute-ready but gated** on:
- **B6 — NAP details** (name/address/phone) for signatures, citations, GBP. See
  `../../status.md` and `app/data/nap.ts`.
- **B5 — Backlinks subscription** (optional) to (a) export competitors' actual
  referring domains for a richer prospect list and (b) monitor link velocity via
  `backlinks/timeseries_summary`. The list here is built from observable competitor
  link-*assets* and known SA link sources; B5 would sharpen it.
- **GBP creation** (owner login) — the anchor citation; see `../local-kit/`.
